A rendering engine built from scratch, as a personal project in my own time, to explore and learn Voxel Global Illumination. I used C++ and OpenGL to implement a set of papers on how to achieve 3D rasterization with the GPU to create Voxel approximations of 3D geometry and store them in fast look up data structures.

This achieves a voxel approximation of the underlying geometry:
Which works at custom levels of resolution:
Its per-pixel efficiency can be improved with better data structures. In the following image a grey scale is used to visualize the per pixel iterations done by the GPU (white represents 100 iterations, black 0 iterations)
Which allows for real time rendering of illumination effects, such as soft shadows, by allowing for efficient data searches for collision detection.
